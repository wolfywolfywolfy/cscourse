The Web: User Experience and User Interface

User Experience

User experience is...

There is a focus on (1) Utility, (2) Ease of Use, (3) Effeciency.
Or in other words, the user's experience when interacting with
the website in any form.

Some things to wonder about, are: is the user enjoying using
the website? Are they actually able to accomplish what they
need to?

One has to also keep stakeholders in mind...things like:
branding, marketing, visual design, and usability are
very important for stakeholders, so one should keep
them in mind when programming (they'll make you keep
them in mind!).

Sources: Wikipedia: UX

User Interface

User Interface is...

There is a focus on the human's ability to interact with the
machine. They should enjoy using the website, which means it
needs to be usable! (No broken screens here!)

Some important aspects to consider are:

  1. Clarity
  2. Concision
  3. Familiarity
  4. Responsiveness
  5. Consistency
  6. Aesthetics
  7. Efficiency
  8. Forgiveness

(Note: Thanks Wikipedia for this awesome list.)

My favorite listed, is forgiveness. A customer should not be punished
by clicking the wrong button, and everything being erased.

Sources: Wikipedia: UI

Web Developement

Web Developement is...

It's not only focused on the looks -- there are also databases and
servers that need to be taken care of. Not to mention, security!

Tim Berners-Lee was working at CERN when he first thought of
linking the idea of hypertext to Transmission Control Protocol with
Domain Name System. Then the World Wide Web was born, with many
webpages. :D
